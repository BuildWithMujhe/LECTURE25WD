# CSS 2D Transforms

This repository contains my practice work and notes from **Lecture 25** of the Web Development Course. In this lecture, I explored CSS 2D Transforms and learned how to manipulate and transform web elements to create dynamic and visually engaging user interfaces.

CSS Transforms allow developers to move, rotate, resize, and reshape elements without affecting the document flow, making modern UI design more interactive and flexible.

---

## 📚 Topics Covered

* Introduction to CSS 2D Transforms
* Translate
* Rotate
* Scale
* Skew
* Transform Origin
* Combining Multiple Transforms
* Visual Effects and Layout Manipulation
* Interactive UI Design

## 🎯 What I Learned

Through this lecture, I learned:

* How CSS transforms modify element appearance
* How `translate()` moves elements
* How `rotate()` changes element orientation
* How `scale()` resizes elements
* How `skew()` creates angled effects
* How transform origin changes transformation behavior
* How multiple transforms can be combined for advanced effects
* Best practices for building interactive interfaces
